=head1 NAME

inc::Package - inc::Module::Install on Kool-aid. Oh Yeah!

=head1 SYNOPSIS

Most C<Makefile.PL> files can contain the single line:

    use inc::Package;

=head1 DESCRIPTION

This is a L<Module::Install> enhancer module. It basically invokes
L<inc::Module::Install> for you and then calls L<Module::Install::Package>'s
C<pkg> function for you.

See L<Module::Install::Package>, for the lovely benefits.
